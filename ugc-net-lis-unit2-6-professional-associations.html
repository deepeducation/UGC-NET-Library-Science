ILA — महत्वपूर्ण तथ्य

ILA की स्थापना 1933 में हुई। यह India की सबसे पुरानी और सबसे बड़ी Library Association है। National Library Day = 12 August = Ranganathan के जन्मदिन (9 Aug 1892) के निकट। ILA का मुख्य journal = ILA Bulletin। ILA ने केंद्रीय Library Act के लिए लगातार Advocacy की है।

✓निष्कर्ष (Conclusion)

Professional Associations Library Profession की Backbone हैं। ALA (1876) से CILIP (2002) तक — ये संस्थाएँ Standards Set करती हैं, Advocacy करती हैं और Library Professionals को एक Platform देती हैं।

भारत में ILA (1933), IASLIC (1955) और IATLIS (1969) ने मिलकर LIS Profession को एक संगठित रूप दिया है। UGC और RRRLF ने Library Development को Institutional Support दिया। UNESCO और IFLA ने International Framework provide किया।

UGC NET की दृष्टि से — प्रत्येक Association का स्थापना वर्ष, HQ, और प्रमुख Publication याद रखना अत्यंत आवश्यक है। इन्हें एक Table के रूप में याद करने का प्रयास करें।

❓FAQs — 10 महत्वपूर्ण प्रश्न

Q1. विश्व की सबसे पुरानी Library Association कौन सी है? इसकी स्थापना किसने की?+
ALA (American Library Association) = विश्व की सबसे पुरानी Library Association। स्थापना = 1876। HQ = Chicago + Washington DC, USA। Co-founder = Melvil Dewey (DDC Inventor)। Significant: ALA, DDC (Dewey Decimal Classification) और Library Journal — तीनों 1876 में ही शुरू हुए। ALA = World's Largest Library Association (Members)। Notable Activities: Banned Books Week, Newbery Medal, Caldecott Medal, ALA Code of Ethics। Washington DC Office = Congress Advocacy।
Q2. IFLA क्या है? इसकी स्थापना कब और कहाँ हुई? India से इसका क्या सम्बन्ध है?+
IFLA (International Federation of Library Associations and Institutions): स्थापना = 1927, Edinburgh, Scotland। HQ = The Hague, Netherlands। 140+ Countries की Members। Annual Conference = WLIC (World Library and Information Congress)। Publications: IFLA Journal। Key Documents: IFLA Code of Ethics (2012), IFLA Internet Manifesto (2002, 2014), UNESCO Public Library Manifesto (with UNESCO, 1994, 2022)। India से सम्बन्ध: ILA + IASLIC = IFLA Members। India में WLIC = 1992 (New Delhi)। Libraries in UN SDGs = IFLA का Primary Advocacy Theme। UNESCO के साथ Joint Manifesto।
Q3. ILA और IASLIC में क्या अंतर है?+
ILA (Indian Library Association): स्थापना = 1933। HQ = New Delhi। Scope = All Libraries (Public, Academic, Special, School)। India की सबसे पुरानी और सबसे बड़ी Library Association। Publication = ILA Bulletin। National Library Day = 12 August। IFLA Member। IASLIC (Indian Association of Special Libraries and Information Centres): स्थापना = 1955। HQ = Kolkata। Scope = Special Libraries + Academic Libraries। Publication = IASLIC Bulletin। Award = Ranganathan Award। IFLA Member। Key Difference: ILA = General (All Libraries)। IASLIC = Specialist (Special Libraries)।
Q4. CILIP का गठन कैसे हुआ? इसकी Chartered Membership क्या है?+
CILIP (Chartered Institute of Library and Information Professionals): स्थापना = 2002। Formation = Library Association (LA, 1898) + Institute of Information Scientists (IIS, 1958) = CILIP 2002। HQ = London, UK। Chartered Membership (MCLIP): MCLIP = Member of CILIP = UK में Library Professional की Standard Qualification। FCLIP = Fellow = Higher Grade। Certification के लिए: Portfolio of Evidence + Professional Development। Awards: Carnegie Medal (Children's Literature) + Kate Greenaway Medal (Illustration)। Publication = CILIP Update। Library Association (1898) = UK की पहली Professional Library Body।
Q5. ASLIB क्या है? इसका पूरा नाम क्या है?+
ASLIB: पूरा नाम = Association of Special Libraries and Information Bureaux। स्थापना = 1924। HQ = London, UK। अब Official नाम = Association for Information Management। Focus: Information Management, Special Libraries, Corporate Information। Publication = Aslib Proceedings (अब Journal of Information Science)। Journal of Documentation से Associated। UK में CILIP से पुरानी (1924 vs Library Association 1898)। Industrial और Corporate Libraries Focus। India में IASLIC इससे Inspired।
Q6. SLA क्या है और इसका IASLIC से क्या सम्बन्ध है?+
SLA (Special Libraries Association): स्थापना = 1909। HQ = Alexandria, Virginia, USA। Focus: Special, Corporate, Government, Medical, Legal Libraries। Publication = Information Outlook। Rebranded = "SLA: Connecting Knowledge Leaders"। Annual SLA Conference। IASLIC से सम्बन्ध: IASLIC = India का SLA Equivalent। दोनों Special Libraries के लिए। IASLIC = 1955 (India)। SLA = 1909 (USA)। IASLIC ने SLA के Model को India में Adapt किया। दोनों IFLA Members। SLA = Corporate/Business Libraries Strong। IASLIC = STI + Academic Libraries Strong।
Q7. RRRLF क्या है? UGC से यह कैसे अलग है?+
RRRLF (Raja Rammohan Roy Library Foundation): स्थापना = 1972। HQ = Kolkata। Under = Ministry of Education (HRD)। Focus = Public Library Development। Named after = Raja Rammohan Roy (1772–1833) = "Father of Modern India"। Functions: State Govts को Financial Assistance, Book Grant Scheme, Model Public Library Scheme, Staff Training। UGC (University Grants Commission): स्थापना = 1956। HQ = New Delhi। Under = Ministry of Education। Focus = Higher Education + Academic Libraries। Functions: University Grants, INFLIBNET, UGC NET, e-Shodh Sindhu, N-LIST। Key Difference: RRRLF = Public Libraries। UGC = Academic/University Libraries।
Q8. UNESCO का Library Science में क्या योगदान है?+
UNESCO (United Nations Educational, Scientific and Cultural Organization): स्थापना = 16 November 1945। HQ = Paris, France। Library Science में Major Contributions: (1) Public Library Manifesto (1949, 1994, 2022) = Free, Open Libraries। (2) School Library Manifesto (1999, 2023)। (3) Memory of the World Programme (1992) = Cultural Heritage। (4) Prague Declaration (2003) = IL as Human Right। (5) Alexandria Proclamation (2005) = IL + Lifelong Learning। (6) World Book Day = 23 April। (7) India में: Delhi Public Library (1951) = First UNESCO-assisted Public Library in South Asia। (8) Libraries in UN SDGs (Sustainable Development Goals)।
Q9. IATLIS क्या करती है? इसका LIS Education में क्या Role है?+
IATLIS (Indian Association of Teachers of Library and Information Science): स्थापना = 1969। Focus = LIS Teachers और Educators। Publication = IATLIS Annals। Role in LIS Education: (1) LIS Curriculum Development में Input। (2) UGC NET LIS Syllabus में भूमिका। (3) Annual National Conference = LIS Education Issues Discuss। (4) LIS Education Quality Standards। (5) International LIS Bodies से Coordination। (6) Research in LIS Education। India में LIS Programs: BLISc (1 year), MLISc (1-2 years), MPhil, PhD। 100+ Universities में LIS Programs। IATLIS = LIS Teaching Community की आवाज।
Q10. All Associations के स्थापना वर्ष को Chronological Order में कैसे याद करें?+
Chronological Order याद करें: (1) 1876 — ALA (Chicago, USA — Oldest, Melvil Dewey, DDC same year)। (2) 1898 — Library Association (UK — बाद में CILIP बनी)। (3) 1909 — SLA (USA, Special Libraries)। (4) 1924 — ASLIB (London, Special Libraries & Info Bureaux)। (5) 1927 — IFLA (Edinburgh → The Hague)। (6) 1933 — ILA (New Delhi — India)। (7) 1945 — UNESCO (Paris)। (8) 1955 — IASLIC (Kolkata — India)। (9) 1956 — UGC (New Delhi)। (10) 1958 — IIS (UK — बाद में CILIP बनी)। (11) 1969 — IATLIS (India)। (12) 1972 — RRRLF (Kolkata)। (13) 2002 — CILIP (LA 1898 + IIS 1958)। Memory Trick: 88-19-24-27-33-45-55-56-58-69-72-2002।
