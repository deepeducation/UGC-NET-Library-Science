📚परिचय और वर्गीकरण

UGC NET Library & Information Science (Code 59) के Unit II का दूसरा Topic है — पुस्तकालयों के प्रकार (Types of Libraries)। परीक्षा में इस Topic से 3–5 प्रश्न नियमित आते हैं।

पुस्तकालयों को उनके उद्देश्य (Purpose), उपयोगकर्ता (Users), संग्रह (Collection), नियंत्रण (Control) और Service Area के आधार पर वर्गीकृत किया जाता है। प्रत्येक प्रकार का पुस्तकालय अपने विशिष्ट Community की Information Needs को पूरा करता है।

⭐Featured Snippet

🎯 Google Featured Snippet

पुस्तकालयों के प्रकार: (1) Academic Library = University/College Libraries — शिक्षा सहायता। (2) Public Library = सभी नागरिकों के लिए Free Access। (3) Special Library = Specific Subject/Organisation — NISCAIR, DESIDOC। (4) National Library = Deposit Library, National Bibliography — Kolkata। (5) School Library = Primary/Secondary Students। (6) Digital Library = Online Resources — NDL India। प्रमुख भारतीय: National Library (Kolkata), Parliament Library (Delhi), Khuda Bakhsh Oriental Library (Patna)।

🎓Academic Library — शैक्षणिक पुस्तकालय

🎓

Academic Library — शैक्षणिक पुस्तकालय

Universities, Colleges, Institutes of Higher Education में स्थापित पुस्तकालय
Higher Education

Academic Libraries उन पुस्तकालयों को कहते हैं जो Higher Educational Institutions — University, College, Technical Institute, Professional School — में स्थापित होते हैं। इनका मुख्य उद्देश्य है — Teaching, Learning और Research को Support करना।

IIT Delhi Library, IISc Bangalore Library, JNU Library (New Delhi), BHU Library (Varanasi), Delhi University Library System (DULS), Anna University Library (Chennai), Jadavpur University Library

UGC NET Tip Academic Library = University + College Libraries। Primary Purpose = Teaching, Learning, Research। Funded by UGC। INFLIBNET = e-Shodh Sindhu + Shodhganga Academic Libraries के लिए। University Library = Largest Academic Library। Departmental Library = Specific Department के लिए (Branch Library)।

🏛️Public Library — सार्वजनिक पुस्तकालय

🏛️

Public Library — सार्वजनिक पुस्तकालय

सभी नागरिकों के लिए Free, Open Access — Tax-supported — No Discrimination
Free for All

Public Library वह पुस्तकालय है जो सभी नागरिकों को बिना किसी Discrimination (जाति, धर्म, शिक्षा, आय) के Free Access प्रदान करता है। यह Public Tax से Funded होता है और Democratic Society की एक Core Institution है।

UNESCO Public Library Manifesto 1994/2022

UNESCO Public Library Manifesto के अनुसार Public Library की 5 Core Missions: (1) Information Services, (2) Education Support, (3) Cultural Development, (4) Community Development, (5) Economic Development। Key Principle: "Free Access to All without Restriction"। No Censorship।

🎯 उद्देश्य
  • Universal Access to Information
  • Lifelong Learning
  • Community Development
  • Cultural Preservation
  • Democratic Participation

  • Children, Elderly

  • Minorities, Immigrants
  • Students, Professionals

  • General Books (Fiction/Non-Fiction)
  • Newspapers, Magazines
  • Reference Books
  • Children's Section
  • Local History Collections
💰 Funding
  • Government (State/Local)
  • Municipal Taxes
  • RRRLF Grants (India)
  • Library Cess (where Act exists)
  • Donations
📋 Sub-types
  • Central Public Library
  • Branch Library

  • Village Library
  • Children's Library
🔧 Special Services
  • Reading Programs
  • Story-telling (Children)
  • Home Delivery (Elderly)
  • e-Resources, Wi-Fi
  • Community Events
🇮🇳 भारतीय उदाहरण

Delhi Public Library (1951), Calcutta Public Library (1836 — India's first), Bombay Public Library, Madras State Central Library (1896), Kerala State Central Library (Thiruvananthapuram), Connemara Public Library (Chennai, 1896)

UGC NET Tip Public Library = Free + Open to All + Tax-supported। India की पहली Public Library = Calcutta Public Library (1836)। RRRLF = Public Library का Financial Support। UNESCO Manifesto = 1949, 1994, 2022। Connemara Public Library (Chennai) = IFLA member। Delhi Public Library = 1951, South Asia में पहली UNESCO-assisted।

🔬Special Library — विशेष पुस्तकालय

🔬

Special Library — विशेष पुस्तकालय

Specific Subject, Organisation या Professional Group के लिए — Depth over Breadth
Subject Specific

Special Libraries किसी विशिष्ट विषय, संस्था या व्यावसायिक समूह की Information Needs को पूरा करने के लिए स्थापित होती हैं। ये Depth में Specialised होती हैं, Breadth में नहीं। Corporate, Government, Research Institutes, Hospitals आदि में पाई जाती हैं।

🎯 विशेषताएँ
  • Subject Specialisation
  • Highly Selective Collection
  • Limited but Specialised Users
  • Current Awareness Priority
  • SDI Services (Personalised)
📋 Sub-types
  • Government Special Libraries
  • Corporate/Industrial Libraries
  • Medical Libraries
  • Legal Libraries
  • News Agency Libraries
🔧 Key Services
  • SDI (Selective Dissemination)
  • CAS (Current Awareness)
  • Literature Search
  • Patent Search
  • Competitive Intelligence
🇮🇳 भारतीय उदाहरण (Special Libraries)

NISCAIR (CSIR STI) · DESIDOC (DRDO Defence) · Parliament Library (Political/Legal) · RBI Library (Economics/Finance) · ICMR Library (Medical Research) · Supreme Court Library (Legal) · NIC Library (IT) · AIIMS Library (Medical)

UGC NET Tip Special Library = Depth + Specialisation + Limited Users। NISCAIR (STI, CSIR) और DESIDOC (Defence, DRDO) = India के सबसे prominent Special Libraries। SLA = Special Libraries Association (USA, 1909)। SDI और CAS = Special Libraries की Signature Services। Parliament Library = India की सबसे बड़ी Special Library।

🌐National Library — राष्ट्रीय पुस्तकालय

🌐

National Library — राष्ट्रीय पुस्तकालय

Deposit Library · National Bibliography · Cultural Heritage Preservation · National Pride
Deposit Library

National Library एक देश का Flagship Library होता है। इसकी प्रमुख विशेषता है Legal Deposit (Statutory Deposit) — देश के सभी Publishers को अपनी Publications की एक या अधिक Copies यहाँ Submit करनी होती हैं। यह देश की National Bibliography तैयार करता है।

🎯 मुख्य कार्य
  • Legal Deposit — सभी Publications
  • National Bibliography तैयार करना
  • Cultural Heritage Preservation
  • National Union Catalogue
  • Interlibrary Loan Hub
📚 विशेषताएँ
  • Largest Collection in Country
  • Comprehensive Coverage
  • Rare & Manuscript Collections
  • International Exchange
  • Research-level Service
🌍 विश्व के प्रमुख
  • Library of Congress (USA, 1800)
  • British Library (UK, 1753/1973)
  • Bibliothèque nationale (France, 1368)
  • Russian State Library
  • National Diet Library (Japan)
🇮🇳 National Library of India

HQ: Kolkata (Belvedere Estate) · 1836 (Calcutta Public Library) → 1891 (Imperial Library) → 1948 (National Library of India) · Under: Ministry of Culture · India की Largest Physical Library · Delivery of Books Act 1954 = Publishers को 3 Copies · National Bibliography of India प्रकाशित करती है · Collection: 22 Lakh+ Items

UGC NET Tip National Library = Deposit Library + National Bibliography। India: National Library = Kolkata (1948)। Under Ministry of Culture (NOT Education)। Delivery of Books Act 1954 = 3 Copies Publishers दें। Library of Congress (USA) = विश्व की Largest Library (170 Million+ Items)। British Library (London) = 1973 में British Museum Library से Separate हुई।

📖School Library — विद्यालय पुस्तकालय

📖

School Library — विद्यालय पुस्तकालय

Primary / Secondary Schools में — Reading Culture · Curriculum Support · Learning Resource Centre
K-12 Education

School Library Primary (Classes 1–8) और Secondary (Classes 9–12) Schools में स्थापित पुस्तकालय है। इसका मुख्य उद्देश्य है — पाठ्यक्रम Support, Reading Habit विकास और Students में Information Literacy बढ़ाना।

🎯 उद्देश्य
  • Curriculum Support
  • Reading Habit Development
  • Information Literacy
  • Imagination & Creativity
  • Lifelong Learning Foundation
📚 Collection
  • Textbooks & Supplementary
  • Story Books, Comics
  • Reference Books
  • Periodicals, Newspapers
  • AVMaterials, CDs, DVDs
📋 NKC/NML Mandate
  • Every School = Functional Library
  • Trained Library Teacher Mandatory
  • Minimum Books per Student
  • E-resources Integration
  • Library = Learning Resource Centre
🇮🇳 India में School Libraries

NEP 2020 में School Libraries को Learning Resource Centre (LRC) का दर्जा। NKC (2005) ने Trained Library Staff की माँग की। CBSE और State Boards में Library Period Mandatory। Kendriya Vidyalaya Sangathan (KVS) Libraries — Best examples of School Libraries in India।

💻Digital Library — डिजिटल पुस्तकालय

💻

Digital Library — डिजिटल पुस्तकालय

Online Resources · No Physical Walls · 24×7 Access · Global Reach
Virtual Library

Digital Library (Electronic Library / Virtual Library) वह पुस्तकालय है जिसमें Resources Digital Format में होते हैं और Internet के माध्यम से कहीं से भी, कभी भी (24×7) Access किए जा सकते हैं। इसकी कोई Physical Boundary नहीं होती।

✨ विशेषताएँ
  • 24×7 Access
  • No Geographic Barrier
  • Multiple Simultaneous Users
  • Hyperlinks & Multimedia
  • Searchable Full-text
📚 Resource Types
  • E-books, E-journals
  • Theses & Dissertations
  • Video/Audio Lectures
  • Databases, Datasets
  • Digital Manuscripts
🏛️ Standards
  • Dublin Core Metadata
  • OAI-PMH Protocol
  • OAIS (ISO 14721)
  • DOI, Handle System
  • LOCKSS, CLOCKSS
🇮🇳 India के Digital Libraries

NDL India (IIT-KGP, 7 Crore+) · Shodhganga (INFLIBNET, Theses) · e-Shodh Sindhu (E-journals) · NOPR (NISCAIR Online Periodicals) · Europeana-equivalent India (in development) · Digital Library of India (DLI) (CMC + IISc)

UGC NET Tip पहला Digital Library Project = Project Gutenberg (1971, Michael Hart)। Digital Library Platforms India: NDL India = IIT Kharagpur। Digital Library Software: DSpace, Greenstone, EPrints, Fedora। Digital Library = Virtual Library = Electronic Library (synonyms)। OAIS (Open Archival Information System) = ISO 14721:2012 = Digital Preservation Standard।

📋 परीक्षा के लिए अतिमहत्वपूर्ण बिंदु (UGC NET Exam Notes)
  • Library Types (6): Academic (Teaching/Research), Public (Free/All/Tax), Special (Subject-specific), National (Deposit/National Bibliography), School (Curriculum/K-12), Digital (Online/24×7)। याद रखें: A-P-S-N-S-D।
  • Public Library: Free + Open to All + Tax-supported। UNESCO Manifesto = 1949, 1994, 2022। 5 Core Missions: Information, Education, Culture, Community, Economic। Calcutta Public Library (1836) = India's First। Delhi Public Library (1951) = First UNESCO-assisted in South Asia।
  • National Library of India: Kolkata (Belvedere)। 1836→1891→1948। Ministry of Culture। Delivery of Books Act 1954 = 3 Copies। India's Largest Physical Library। National Bibliography of India।
  • Special Libraries India: NISCAIR (STI, CSIR, Delhi)। DESIDOC (Defence, DRDO, Delhi)। Parliament Library (1921, Delhi — Largest Special Library India)। NASSDOC (Social Science, ICSSR, 1969)। Khuda Bakhsh (Oriental, Patna, 1891)।
  • Connemara Public Library (Chennai, 1896): Deposit Library + Public Library। IFLA Member। Tamil Heritage। Legal Deposit for South India।
  • Asiatic Society of Bengal (Kolkata, 1784): William Jones ने Founded। India's Oldest Library। Oriental Manuscripts।
  • Academic Library Specifics: University Library = Largest Academic Library। Departmental Library = Branch। INFLIBNET = Academic Library Network। e-Shodh Sindhu = E-resources for 400+ Universities।
  • Digital Library: Project Gutenberg (1971) = पहला। NDL India = IIT Kharagpur। Software: DSpace, Greenstone, EPrints, Fedora। Standards: Dublin Core, OAI-PMH, OAIS (ISO 14721)।
  • School Library: NKC + NML = Every School must have Trained Library Staff। NEP 2020 = Library as Learning Resource Centre। KVS Libraries = Best Model। AASL (American Association of School Librarians)।
  • Library of Congress (USA): World's Largest Library। 170 Million+ Items। 1800। LC Classification System (LCC)। CIP (Cataloguing in Publication)। British Library = 1753/1973, London।

✓निष्कर्ष (Conclusion)

पुस्तकालयों के विभिन्न प्रकार — Academic, Public, Special, National, School और Digital — प्रत्येक अपने Community की विशिष्ट Information Needs को पूरा करते हैं। कोई एक प्रकार दूसरे से श्रेष्ठ नहीं — सभी मिलकर एक देश की Knowledge Infrastructure बनाते हैं।

भारत में पुस्तकालयों का विकास National Library (Kolkata) से लेकर NDL India (Digital) तक हुआ है। RRRLF, INFLIBNET, NKC और NML जैसी Initiatives ने इस Network को मजबूत किया है। विशेष रूप से NISCAIR, DESIDOC जैसी Special Libraries ने India के Scientific और Defence Research में अमूल्य योगदान दिया है।

❓FAQs — 10 महत्वपूर्ण प्रश्न

Q1. पुस्तकालयों के प्रकार कितने हैं? प्रत्येक का मुख्य उद्देश्य क्या है?+
पुस्तकालयों के 6 प्रमुख प्रकार (A-P-S-N-S-D): (1) Academic = University/College — Teaching, Learning, Research। (2) Public = सभी नागरिक — Free, Open, Community Service। (3) Special = Specific Subject/Org — Depth, SDI, CAS। (4) National = Deposit Library — National Bibliography, Heritage। (5) School = Primary/Secondary — Curriculum Support, Reading Habit। (6) Digital = Online — 24×7, No Geographic Barrier। अन्य: Medical, Legal, Corporate, Government, Agricultural, News Libraries। प्रत्येक की Funding, Users, Collection और Services अलग होती हैं।
Q2. Public Library की परिभाषा क्या है? UNESCO Manifesto क्या कहता है?+
Public Library = वह पुस्तकालय जो सभी नागरिकों को बिना किसी Discrimination के Free Access देता है, जो Public Tax से Funded होता है। UNESCO Public Library Manifesto: First = 1949, Revised = 1994, Latest = 2022। Key Principles: Free Access to All, No Censorship, Tax-supported, Open without Restriction। 5 Core Missions (1994): (1) Information Services, (2) Education Support, (3) Cultural Development, (4) Community Development, (5) Economic Development। India की पहली Public Library = Calcutta Public Library (1836)। Delhi Public Library (1951) = First UNESCO-assisted, South Asia।
Q3. National Library of India के बारे में महत्वपूर्ण तथ्य क्या हैं?+
National Library of India: HQ = Kolkata, Belvedere Estate। Evolution: 1836 (Calcutta Public Library) → 1891 (Imperial Library under Lord Curzon) → 1948 (National Library of India)। Under: Ministry of Culture (NOT Education)। India की Largest Physical Library। Delivery of Books Act 1954 = Publishers को 3 Copies देनी होती हैं। National Bibliography of India Publish करती है। Collection: 22 Lakh+ Items। Legal Deposit Library for India। International: Library of Congress (USA, 1800, 170 Million+) = World's Largest। British Library (London, 1753/1973)।
Q4. Special Library और Academic Library में मुख्य अंतर क्या है?+
Academic Library: University/College में। Users = Students + Faculty + Researchers। Collection = Broad Subject Coverage (Curriculum-based)। Purpose = Teaching, Learning, Research। Funded by University/UGC। Services = Reference, ILL, Online DBs। Special Library: Specific Organization में। Users = Limited, Specific (only that Org staff)। Collection = Highly Specialised (Depth not Breadth)। Purpose = Specific Subject/Organisation needs। Funded by Parent Organisation। Services = SDI, CAS, Patent Search, Competitive Intelligence। Key Difference: Academic = Broad, Special = Deep। India examples: Academic = IIT Library। Special = NISCAIR (STI), DESIDOC (Defence), Parliament Library।
Q5. NISCAIR और DESIDOC क्या हैं? ये किस प्रकार की Libraries हैं?+
NISCAIR (National Institute of Science Communication and Information Resources): New Delhi। CSIR के अंतर्गत। 2002 में INSDOC + NISCOM का Merger। Type: STI Special Library + Publisher। 17 CSIR Journals Publish करता है। Science Reporter (Popular Science)। NISCAIR Online Periodicals Repository (NOPR)। DESIDOC (Defence Scientific Information and Documentation Centre): New Delhi। DRDO के अंतर्गत। Type: Defence Special Library। Annals of Library and Information Studies (ALIS) Publish करता है। DRDO Scientists को STI Services। यह दोनों India के सबसे प्रसिद्ध Special Libraries हैं।
Q6. Connemara Public Library क्या है? इसे Deposit Library क्यों कहते हैं?+
Connemara Public Library: Chennai (Tamil Nadu)। स्थापना = 1896। Named after Lord Connemara (Madras Governor)। यह एक Public Library + National Deposit Library दोनों है। Deposit Library कहलाती है क्योंकि Delivery of Books Act 1954 के अनुसार South India के Publishers को अपनी Publications की एक Copy यहाँ देनी होती है। IFLA (International Federation of Library Associations) का Member है। Tamil heritage collection। दक्षिण भारत की सबसे महत्वपूर्ण Public Library। India में 4 Deposit Libraries हैं: National Library (Kolkata), National Library (Delhi — Parliament), Connemara (Chennai), Central Library (Mumbai)।
Q7. Digital Library क्या है? इसकी प्रमुख विशेषताएँ कौन सी हैं?+
Digital Library (Electronic Library / Virtual Library) = वह Library जिसमें Resources Digital Format में हों और Internet से 24×7 Access किए जा सकें। Features: (1) No Physical Boundary, (2) 24×7 Access, (3) Multiple Simultaneous Users, (4) Searchable Full-text, (5) Hyperlinks & Multimedia, (6) Global Reach। पहला Digital Library = Project Gutenberg (1971, Michael Hart)। India: NDL India (IIT Kharagpur) = 7 Crore+। Software: DSpace, Greenstone, EPrints, Fedora। Standards: Dublin Core Metadata, OAI-PMH Protocol, OAIS (ISO 14721:2012) = Preservation Standard। Digital vs Physical Library: Physical = Location-bound, Time-bound। Digital = Location-free, Time-free।
Q8. Khuda Bakhsh Oriental Library क्या है? इसे National Library का दर्जा क्यों दिया गया?+
Khuda Bakhsh Oriental Public Library: Patna, Bihar। स्थापना = 1891 by Maulvi Khuda Bakhsh Khan। Collection: 21,000+ Rare Arabic, Persian, Urdu, Mughal Manuscripts। Mughal Miniature Paintings। पहले Khuda Bakhsh के पिता Muhammad Bakhsh ने 1,400 Manuscripts से शुरुआत की। National Importance: India के Parliament ने Khuda Bakhsh Oriental Public Library Act 1969 पारित करके इसे National Importance का Institution घोषित किया। यह Ministry of Culture के अंतर्गत है। इसकी Manuscripts Collection UNESCO Memory of the World Programme में शामिल हैं।
Q9. Parliament Library के बारे में क्या जानते हैं?+
Parliament Library (Sansad Library): New Delhi। स्थापना = 1921 (Central Legislative Assembly के साथ)। Type = Special Library (Government)। India की सबसे बड़ी Special Library। Users = Members of Parliament (MPs और MLAs), Parliamentary Staff, Researchers। Collection: Political Science, Constitutional Law, Indian History, Economics, Parliamentary Debates (Hansard), State Assembly Proceedings। Services: Reference, Research Support, Current Awareness। Notable: Parliamentary Research Services provides Analytical Briefs to MPs। Collection: 10 Lakh+ volumes। Under: Lok Sabha Secretariat। LARRDIS (Library and Reference, Research, Documentation and Information Service) = Parliament Library का Research Wing।
Q10. School Library और Academic Library में क्या अंतर है? NEP 2020 में School Libraries का क्या स्थान है?+
School Library: Primary/Secondary Schools (K-12)। Users = School Students + Teachers। Collection = Textbooks, Story Books, Reference, AV Materials। Purpose = Curriculum Support + Reading Habit + Imagination। Academic Library: Higher Education। Users = College/University Students + Faculty + Researchers। Collection = Advanced Academic + Research Resources। Purpose = Teaching, Learning, Research। NEP 2020 में School Libraries को "Learning Resource Centre (LRC)" का दर्जा। Importance: (1) Critical thinking develop होती है। (2) Self-learning promote होती है। NKC (2005) = Every school में Trained Library Staff mandatory। AASL (American Association of School Librarians) Standards। India में KVS (Kendriya Vidyalaya Sangathan) Libraries = School Libraries का Best Model।
