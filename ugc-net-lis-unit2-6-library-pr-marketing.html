📢परिचय (Introduction)

UGC NET Library & Information Science (Code 59) के Unit II का छठा Topic है — पुस्तकालय में जनसंपर्क (Public Relations in Libraries)। परीक्षा में इस Topic से 2–4 प्रश्न आते हैं।

Library Public Relations (PR) वह सुनियोजित प्रयास है जिसके द्वारा Library अपने Users, Funding Bodies, Community और General Public के साथ सकारात्मक सम्बन्ध बनाती है। Ranganathan के Five Laws का Law 1 ("Books are for use") और Law 2 ("Every reader his book") PR की नींव हैं।

Digital Age में Library PR का दायरा बढ़ गया है — Social Media, Library Websites, E-Newsletters, Blogs — सभी Modern PR Tools बन गए हैं।

Library Public Relations क्या है? Library PR = Library और उसके Users/Community के बीच आपसी समझ और सकारात्मक संबंध बनाने का सुनियोजित प्रयास। IPRA (1955) Definition: "PR is a management function which evaluates public attitudes, identifies policies and procedures of an individual or organization with public interest, and executes a programme of action to earn public understanding and acceptance." 📊Library Marketing — 4P Marketing Mix

📊

Library Marketing — 4P Marketing Mix

Philip Kotler — Marketing for Non-profit Organizations | Library = Service Organization
Philip Kotler

Library Marketing = Library की Services को Users तक पहुँचाने की सुनियोजित Strategy। Philip Kotler ने Marketing को Non-profit Organizations पर Apply किया। Library Marketing की नींव Ranganathan के Law 2 और 3 में है।

P
Product (सेवा)
Library की Services = Product। Reference, ILL, E-resources, Digital Library। Service Quality ही Product।
P
Price (शुल्क)
Library में Price = User का Time + Effort + Membership Fee। Free Services = Price = 0। Value Proposition।
P
Place (स्थान)
Library Location। Opening Hours। Branch Libraries। Online Access = Virtual Place। Mobile Library।
P
Promotion (प्रचार)
PR, Publicity, Social Media, Events, Newsletters। Service को Users को बताना।

4P से 7P — Extended Marketing Mix (Services)

5th P — People
  • Library Staff = Service Delivery
  • Trained, Courteous Staff
  • Professional Librarians
  • Customer Service Skills
6th P — Process
  • How Services are Delivered
  • OPAC Search Process
  • Book Issue/Return Process
  • Reference Interview Process
7th P — Physical Evidence
  • Library Building Design
  • Signage, Furniture
  • Website Design
  • Library Atmosphere/Environment
Philip Kotler — Marketing Guru

Philip Kotler = "Father of Modern Marketing"। उन्होंने "Marketing for Non-profit Organizations" (1975) में Library Marketing का Framework दिया। Kotler का मानना था — Library को अपनी Services को User Needs के आधार पर Design और Promote करना चाहिए — Librarian की Convenience के लिए नहीं। Kotler's 4P = Product + Price + Place + Promotion = Marketing Mix।

UGC NET Tip 4P Marketing Mix = Product + Price + Place + Promotion। Philip Kotler = "Father of Modern Marketing"। Library में Product = Services। Price = User's Effort/Time (mostly Free)। Place = Location + Online Access। Promotion = PR + Social Media + Events। 7P = 4P + People + Process + Physical Evidence।

✓निष्कर्ष (Conclusion)

Library Public Relations केवल Library का "Advertising" नहीं है — यह एक Planned, Sustained और Two-way Communication Process है जो Library और उसकी Community के बीच Mutual Understanding और Trust बनाती है।

Digital Age में Social Media, Library 2.0 और Mobile Apps ने PR के नए आयाम खोले हैं। Philip Kotler के 4P Marketing Mix ने Library Services को User Needs-centric बनाने में मदद की। Library Advocacy — ILA से IFLA तक — ने Libraries को National और International Policy में अपनी जगह दिलाई है।

Ranganathan के Five Laws की भावना — हर पाठक को उसकी किताब, पाठक का समय बचाएँ — PR और Marketing के माध्यम से ही पूरी होती है। A good Library without good PR is like a hidden treasure — useful only when found.

❓FAQs — 10 महत्वपूर्ण प्रश्न

Q1. Library Public Relations की परिभाषा क्या है? PR और Publicity में क्या अंतर है?+
Library PR = Library और उसके Users, Community, Funding Bodies के बीच Mutual Understanding और Goodwill Build करने का Planned, Sustained, Two-way Communication। IPRA (1955) = "Management function which evaluates public attitudes, identifies policies with public interest, and executes programme to earn public understanding." PR vs Publicity: PR = Long-term, Two-way, Planned, Sustained, Earned Coverage। Publicity = Short-term, One-way, News-based, Media Coverage। Publicity = PR का एक Tool। PR में Publicity include होती है — लेकिन PR बहुत अधिक है Publicity से। Edward Bernays = "Father of PR" = "Engineering of Consent"।
Q2. Library Marketing में 4P Marketing Mix क्या है?+
4P Marketing Mix = Philip Kotler (Father of Modern Marketing)। Library पर Apply: Product = Library Services (Reference, ILL, E-resources, Digital Library — Service Quality = Product)। Price = User का Time + Effort + Membership (mostly Free = Price = 0, यही Library का USP)। Place = Library Location + Opening Hours + Branch Libraries + Online Access (Virtual Place)। Promotion = PR, Social Media, Newsletter, Events, Press Release। Extended 7P: + People (Staff), + Process (Service Delivery), + Physical Evidence (Building, Website Design)। Philip Kotler = "Marketing for Non-profit Organizations" (1975)।
Q3. Library 2.0 क्या है? इसे किसने Coin किया?+
Library 2.0 = Web 2.0 Technology का Library में Application। Term Coined by: Michael Casey (2005) — "LibraryCrunch" Blog पर। Library 2.0 के 4 Components: (1) User-centered Change, (2) Multi-media Experience, (3) Socially Rich, (4) Communally Innovative। Library 2.0 Tools: Blogs (Library News), Wikis (Collaborative), RSS Feeds (Alerts), Podcasts, Social Bookmarking, Tagging/Folksonomy। Social Media: Facebook, Twitter, Instagram, YouTube। Key Concept: Participatory Library = Users = Active Participants, not just Recipients। Folksonomy = User-generated Tags। Taxonomy = Expert-created Classification।
Q4. ROPE और RACE PR Models क्या हैं?+
ROPE Model (Jerry Hendrix): Research (Situation Analysis, SWOT), Objectives (Goals Set करना), Programming (Plan Execute), Evaluation (Results Measure)। Cyclical Process। RACE Model (John Marston, 1963): Research (Facts Gather), Action (Planning), Communication (Execution), Evaluation (Assessment)। दोनों में Evaluation = Final Stage। ROPE और RACE essentially similar steps — Different nomenclature। Library PR Planning = Research → Objectives → Plan → Execute → Evaluate।
Q5. Library Advocacy क्या है? PR से यह कैसे अलग है?+
Library Advocacy = Libraries के लिए Funding, Legislation और Policy Support पाने का Organised प्रयास। Target = Decision Makers, Politicians, Funding Authorities। Goals: Budget बढ़ाना, Library Act बनवाना, Library Closures रोकना। PR से अंतर: PR = General Public को Target (Image Building)। Advocacy = Specific Decision Makers को Target (Policy Change)। Advocacy = PR का Strategic/Political Level। Examples: ALA = Washington DC में Congress को Lobby। IFLA = UNESCO, UN में Advocacy। ILA = Central Library Act की माँग। IFLA Libraries & SDGs = Sustainable Development Goals में Libraries की भूमिका = Major Advocacy Theme। ROI (Return on Investment) = Library Value को Numbers में दिखाना = Advocacy Tool।
Q6. Press Release क्या है? Library PR में इसका क्या उपयोग है?+
Press Release (Press Note) = किसी Organisation द्वारा Media (Newspapers, TV, Online) को जारी की गई Official Written Statement — किसी Important News/Event के बारे में। Library में Use: (1) New Service Launch। (2) Important Event Announcement। (3) New Collection/E-resource। (4) Award या Achievement। (5) Library Inauguration। Format: Headline, Date, Body (Who, What, When, Where, Why), Quotes, Contact Info। Good Press Release = Media Coverage → Free Publicity। यह सबसे Important Traditional PR Tool है। Digital Era में: Press Release + Social Media Post + Blog = Integrated Announcement।
Q7. Library Newsletter क्या है? इसके प्रकार और उपयोग क्या हैं?+
Library Newsletter = Library द्वारा Regular Intervals पर Users को दी जाने वाली Publication (Print या Digital)। Types: (1) Monthly/Quarterly Newsletter = General Updates। (2) New Arrivals List = New Books, Journals। (3) Subject Newsletter = Specific Subject में New Resources। (4) E-Newsletter = Email-based। (5) CAS Bulletin = Current Awareness Service। Contents: New Acquisitions, Events, Database Announcements, Staff News, Articles। Benefits: Regular Communication, User Retention, CAS Tool। Modern: E-Newsletter + Blog + Social Media = Integrated Newsletter। India: NISCAIR, INFLIBNET = Excellent Library Newsletters।
Q8. Library Annual Report का PR में क्या महत्व है?+
Library Annual Report = Library की एक वर्ष की Activities, Achievements, Statistics और Future Plans का Comprehensive Document। PR में Importance: (1) Accountability = Funding Bodies को Library Performance दिखाना। (2) Advocacy Tool = Budget Increase के लिए Evidence। (3) Public Trust = Transparency demonstrate। (4) Benchmarking = Past Performance vs Current। Contents: Collection Statistics, User Statistics, Budget Used, New Services, Events, Challenges, Future Plans। Annual Report = Most Important PR Tool for Advocacy। India में: UGC-recognized Libraries को Annual Reports Submit करनी होती हैं। INFLIBNET Annual Report = Academic Library Statistics।
Q9. SWOT Analysis Library PR में कैसे उपयोगी है?+
SWOT Analysis = Library की Internal और External Factors का Assessment — PR Planning के लिए: Strengths (आंतरिक ताकत): Large Collection, Trained Staff, Free Service, Central Location, E-resources। Weaknesses (कमज़ोरियाँ): Limited Budget, Outdated Collection, Poor PR, Limited Hours, Staff Shortage। Opportunities (अवसर): Digital Services, Social Media Engagement, Partnerships with NGOs, Government Grants, Online Events। Threats (ख़तरे): Internet/Google Competition, Budget Cuts, Non-use, Digital Divide, COVID-type Closures। Use: ROPE/RACE Model में Research Stage। PR Strategy बनाने से पहले SWOT ज़रूरी। Strength को Promote करना = PR। Weakness को Address करना = Service Improvement।
Q10. Library Events और Programs PR में कैसे काम करते हैं? Non-users को कैसे Engage करें?+
Library Events और Programs = Non-users को Library में आकर्षित करने का सबसे प्रभावी तरीका। Types: Book Exhibitions, Reading Clubs, Author Talks, Quiz Competitions, Library Week (National Library Day = 12 Aug), Story-telling (Children), Film Screenings, Workshops। Non-users को Engage करने के लिए: (1) Library से बाहर जाएँ — School Visits, Community Centres। (2) Mobile Library — उनके पास Library लाएँ। (3) Social Media — जहाँ Non-users हैं वहाँ पहुँचें। (4) Partnerships — Schools, NGOs के साथ। (5) Free Events — Entry Barrier खत्म। India: RRRLF = Public Library Events को Fund करती है।
