⭐Featured Snippet

🎯 Google Featured Snippet

Information Literacy क्या है? Information Literacy (सूचना साक्षरता) = वह क्षमता जिससे व्यक्ति यह पहचान सके कि उसे सूचना की ज़रूरत है, उसे ढूँढ सके, Evaluate कर सके और प्रभावी ढंग से Use कर सके। ALA Definition (1989): "An information literate person is able to recognize when information is needed and has the ability to locate, evaluate, and use effectively the needed information." ACRL Standards = 2000 (5 Standards)। ACRL Framework = 2015 (6 Frames)। Prague Declaration = 2003 (UNESCO)।

Information Needs — सूचना की आवश्यकता

Robert S. Taylor (1968) — 4 Levels of Information Need
Taylor's Model

Information Need = वह अवस्था जब किसी व्यक्ति को किसी निर्णय लेने, समस्या सुलझाने या ज्ञान प्राप्त करने के लिए सूचना की आवश्यकता होती है।

Robert S. Taylor (1968) — 4 Levels of Information Need

🤔
Q1 — Visceral Need
अस्पष्ट अनुभव। सूचना की Vague आवश्यकता। व्यक्ति खुद भी स्पष्ट नहीं।
💭
Q2 — Conscious Need
Mental description। अनुभव थोड़ा स्पष्ट हुआ लेकिन Words में नहीं।
🗣️
Q3 — Formalized Need
Rational statement। Words में Express। Librarian को बताने योग्य।
🔍
Q4 — Compromised Need
System को Adapt। Library/Database की Constraints के अनुसार बदला।
📋 Information Need को प्रभावित करने वाले Factors
  • Subject/Discipline — STEM vs Humanities अलग Needs
  • Role — Researcher vs Student vs Professional
  • Seniority — Junior vs Senior Researcher
  • Task at Hand — Research, Decision, Problem
  • Environment — Academic, Corporate, Government
🎯 Information Need vs Information Want vs Information Demand
  • Need = वास्तविक सूचना की ज़रूरत (Objective)
  • Want = जो सूचना चाहिए वो (Subjective)
  • Demand = जो Library से माँगी जाती है
  • Need ≥ Want ≥ Demand
  • Effective User Study = Gap कम करना
UGC NET Tip Robert S. Taylor के 4 Levels: V-C-F-C = Visceral, Conscious, Formalized, Compromised। Taylor = 1968। Visceral = Most Vague। Compromised = Most Specific (System-adapted)। यह Information Science में सबसे Quoted Model है।

User Studies — परिभाषा और विधियाँ

Users की Information Needs, Seeking Behaviour और Library Use को समझने का वैज्ञानिक अध्ययन
Research Method

User Studies (उपयोगकर्ता अध्ययन) Library Research की वह शाखा है जो यह समझने का प्रयास करती है कि Users Information कैसे Seek करते हैं, Library का उपयोग कैसे करते हैं और उनकी Needs क्या हैं।

🎯 उद्देश्य
  • User Needs Identify करना
  • Library Services Improve करना
  • Collection Development Guide करना
  • Non-users को समझना
  • Information Policy बनाना
🔬 Research Methods
  • Questionnaire/Survey — सबसे Common
  • Interview — In-depth
  • Observation — Direct
  • Diary Method — Self-report
  • Citation Analysis
📋 Study के प्रकार
  • Formal User Studies — Scientific
  • Informal User Studies — Observation
  • Macro vs Micro Studies
  • Quantitative vs Qualitative
  • OPAC Log Analysis
पहला User Study

Formal User Studies की शुरुआत 1940s में हुई। Bernal (1948) ने Scientific Information के Social Function पर Study की। Royal Society Scientific Information Conference (1948) = User Studies का Landmark। India में: INSDOC (बाद में NISCAIR) ने STI Users पर Studies कीं।

UGC NET Tip User Studies = User Needs + User Behaviour + Library Use Pattern। सबसे Common Method = Questionnaire/Survey। Bernal (1948) = First Formal User Study। User Study के 2 Approaches: System-centred (Library की दृष्टि से) vs User-centred (User की दृष्टि से)। Modern trend = User-centred।

🔍Information Seeking Behaviour (ISB) Models

ISB Models यह समझाते हैं कि लोग सूचना कैसे ढूँढते हैं — क्या Process अपनाते हैं:

Wilson's Model (1981, 1996)
T.D. Wilson — Most Cited ISB Model
  • Information Need → Aroused
  • Information Seeking → Search begins
  • Information Use → Applied to problem
  • Barriers: Personal, Role, Environment
  • 1996 = Revised with Cognitive model
Kuhlthau's ISP Model (1991)
Carol C. Kuhlthau — Information Search Process
  • Initiation — Task शुरू
  • Selection — Topic Choose
  • Exploration — Search (Confused)
  • Formulation — Focus clear
  • Collection — Information Gather
  • Presentation — Task complete
Ellis's Model (1989)
David Ellis — Behavioural Model
  • Starting — शुरुआती Sources
  • Chaining — References follow
  • Browsing — Semi-directed
  • Differentiating — Filter Sources
  • Monitoring — Awareness
  • Extracting — Relevant info
Dervin's Sense-Making (1983)
Brenda Dervin — Cognitive Gap Model
  • User = Moving through Time-Space
  • Gap = Information Need = Gap in Knowledge
  • Bridge = Information = Fills Gap
  • Outcome = Sense Made
  • User-centred Approach
Marchionini's ISP (1995)
Gary Marchionini — Electronic Environments
  • Recognize & Accept Problem
  • Define & Understand Problem
  • Choose Search System
  • Formulate Query
  • Execute Search
  • Examine Results → Iterate
Zipf's Principle (1949)
George Zipf — Least Effort Principle
  • Principle of Least Effort
  • Users choose Easiest Path
  • Nearest Source = Preferred
  • Explains why Google popular
  • Less effort = More likely to use
UGC NET Tip ISB Models याद करें: Wilson (1981/1996) = Need→Seeking→Use, Most Cited। Kuhlthau (1991) = ISP — 6 Stages (Initiation, Selection, Exploration, Formulation, Collection, Presentation)। Ellis (1989) = 6 Behaviours (Starting, Chaining, Browsing, Differentiating, Monitoring, Extracting)। Dervin (1983) = Sense-Making, Gap-Bridge। Zipf (1949) = Least Effort।

Information Literacy — सूचना साक्षरता

Paul Zurkowski (1974) — Term coined | ALA (1989) — Classic Definition | ACRL (2000, 2015)
Core Competency

Information Literacy (IL) = सूचना की ज़रूरत को पहचानने, उसे ढूँढने, Evaluate करने और प्रभावी ढंग से Use करने की क्षमता। यह 21वीं सदी में Digital Citizenship का आधार है।

पहली बार — Paul Zurkowski (1974)

Paul Zurkowski (President, Information Industry Association) ने 1974 में पहली बार "Information Literacy" शब्द का प्रयोग किया। उन्होंने कहा — जो व्यक्ति Information Techniques का उपयोग करके Problems solve कर सके = Information Literate।

⭐ALA Definition 1989 — क्लासिक परिभाषा

⭐

ALA Presidential Committee Report on Information Literacy, 1989

American Library Association | Most Cited IL Definition | "Recognize, Locate, Evaluate, Use"
Classic Definition

ALA (1989) की परिभाषा: "To be information literate, a person must be able to recognize when information is needed and have the ability to locate, evaluate, and use effectively the needed information."

ALA 1989 के अनुसार Information Literate Person के 4 गुण

1
Recognize
यह पहचानना कि सूचना की आवश्यकता कब है। Problem Identification। Gap in Knowledge।
2
Locate
आवश्यक सूचना को ढूँढना और Access करना। Search Strategies। Source Selection।
3
Evaluate
प्राप्त सूचना की Quality और Relevance Judge करना। Critical Thinking। Source Credibility।
4
Use Effectively
सूचना को अपनी Problem Solve करने में लगाना। Ethical Use। Citation। Application।
ALA (1989) — Final Report Statement

ALA Report (1989) ने कहा: "Information literate people are those who have learned how to learn. They know how to learn because they know how knowledge is organized, how to find information, and how to use information in such a way that others can learn from them." — Information Literate = Self-directed Learner।

UGC NET Tip ALA 1989 IL Definition = Recognize + Locate + Evaluate + Use। R-L-E-U याद करें। Paul Zurkowski (1974) ने Term Coin किया। ALA (1989) ने Classic Definition दी। ACRL (2000) ने Higher Education के लिए 5 Standards बनाए। ACRL (2015) ने Framework (6 Frames) से Replace किया।

📋ACRL IL Standards (2000) और Framework (2015)

📋

ACRL IL Standards, 2000

5 Standards + 22 Performance Indicators + 87 Outcomes
  • Standard 1: Determine extent of information needed। (Need Recognition)
  • Standard 2: Access needed information effectively & efficiently। (Finding)
  • Standard 3: Evaluate information & sources critically। (Evaluating)
  • Standard 4: Use information effectively to accomplish a purpose। (Using)
  • Standard 5: Understand economic, legal, social issues of information use। (Ethics/Legal)
🖼️

ACRL Framework, 2015

6 Frames — Threshold Concepts — Metacognition
  • Frame 1: Authority is Constructed and Contextual
  • Frame 2: Information Creation as a Process
  • Frame 3: Information has Value
  • Frame 4: Research as Inquiry
  • Frame 5: Scholarship as Conversation
  • Frame 6: Searching as Strategic Exploration
Standards vs Framework — मुख्य अंतर

Standards (2000) = Prescriptive, Measurable Outcomes, Checklist Approach, Task-oriented। Framework (2015) = Conceptual, Threshold Concepts, Metaliteracy, Reflective approach। Framework में Learner की Agency और Metacognition पर Focus। Big Idea: Information Literacy = Not a Set of Skills but a Way of Thinking।

User Education — Library Orientation & Instruction

Library Orientation → Library Instruction → Information Literacy Instruction
IL Program

User Education वह प्रक्रिया है जिसके माध्यम से Library Users को Library Resources, Services और Information Skills के बारे में Formally शिक्षित किया जाता है। यह IL का Practical Application है।

User Education के 3 Levels

Level 1 — Library Orientation
  • Library की Physical Tour
  • Where to find What
  • Library Rules, Hours
  • OPAC Introduction
  • New Students के लिए
Level 2 — Library Instruction
  • How to Search Databases
  • Reference Tools use करना
  • Boolean Operators (AND/OR/NOT)
  • Citation Styles
  • Subject-specific Resources
Level 3 — IL Instruction
  • Critical Evaluation of Sources
  • Research Process
  • Plagiarism Awareness
  • Citation Management (Zotero)
  • Data Literacy, Digital Literacy

✓निष्कर्ष (Conclusion)

Library Users की समझ और Information Literacy का विकास — ये दोनों आधुनिक Library Science के Core हैं। Ranganathan का Law 2 ("Every Reader His Book") और Law 4 ("Save the Time of the Reader") Users को Central रखते हैं।

Information Literacy — Paul Zurkowski (1974) से शुरू होकर ALA (1989), ACRL (2000, 2015) और UNESCO Prague Declaration (2003) तक — एक Global Priority बन गई है। 21वीं सदी में IL केवल Library Skill नहीं बल्कि Digital Citizenship का आधार है।

ISB Models (Wilson, Kuhlthau, Ellis, Dervin) ने हमें समझाया कि Information Seeking एक Complex Cognitive Process है। User-centred Library Services, effective IL Programs और robust User Studies — यही भविष्य की Library का मार्ग है।

❓FAQs — 10 महत्वपूर्ण प्रश्न

Q1. Information Literacy की ALA (1989) Definition क्या है? इसे किसने Coin किया था?+
Information Literacy Term = Paul Zurkowski (President, Information Industry Association, USA) ने 1974 में Coined किया। ALA (1989) Classic Definition: "To be information literate, a person must be able to recognize when information is needed and have the ability to locate, evaluate, and use effectively the needed information." — ALA Presidential Committee on Information Literacy, 1989। 4 Core Abilities: Recognize (need पहचानना) + Locate (ढूँढना) + Evaluate (मूल्यांकन) + Use Effectively (प्रभावी उपयोग)। Memory: R-L-E-U।
Q2. ACRL Standards (2000) और ACRL Framework (2015) में क्या अंतर है?+
ACRL IL Competency Standards (2000): 5 Standards + 22 Performance Indicators + 87 Outcomes। Prescriptive, Measurable, Checklist-based। Skills-oriented। 5 Standards: Determine need, Access, Evaluate, Use, Understand issues। ACRL Framework for IL in Higher Ed (2015): Standards को Replace किया। 6 Frames based on Threshold Concepts। 6 Frames: Authority Constructed, Information Creation as Process, Information has Value, Research as Inquiry, Scholarship as Conversation, Searching as Strategic Exploration। Conceptual, not prescriptive। Metaliteracy पर Based। Reflective, Metacognitive approach। Learner's Agency पर Focus।
Q3. Robert S. Taylor के 4 Levels of Information Need क्या हैं?+
Robert S. Taylor (1968) के 4 Levels: (1) Q1 — Visceral Need = Most Vague। अस्पष्ट Feeling। व्यक्ति खुद words में Express नहीं कर सकता। (2) Q2 — Conscious Need = Brain में थोड़ा Formulate हुआ। Mental Description। अभी भी Vague। (3) Q3 — Formalized Need = Rational Statement। Words में Express। Librarian को बता सकते हैं। (4) Q4 — Compromised Need = System-adapted। Library/Database की Constraints के अनुसार Modify किया। Search Query। Memory: V-C-F-C। Q1 = Most Vague → Q4 = Most Specific।
Q4. Kuhlthau का ISP Model क्या है? इसके 6 Stages कौन से हैं?+
Kuhlthau's Information Search Process (ISP) Model, 1991 = Carol C. Kuhlthau। Students के Research Process पर Based। Library Anxiety Phenomenon include। 6 Stages: (1) Initiation = Task मिला, Uncertainty। (2) Selection = Topic Choose, Optimism। (3) Exploration = Search, Confusion/Frustration। (4) Formulation = Focus Clear, Clarity। (5) Collection = Information Gather, Confidence। (6) Presentation = Task Complete, Relief/Satisfaction। Each stage has: Feelings (Affective), Thoughts (Cognitive), Actions (Physical)। Library Anxiety = Exploration Phase में highest।
Q5. Prague Declaration (2003) और Alexandria Proclamation (2005) में क्या अंतर है?+
Prague Declaration (2003): "Towards an Information Literate Society"। National Forum on IL, Prague, Czech Republic। Organized by NCLIS (USA) + UNESCO। Key Statement: "IL is a Basic Human Right"। IL = Essential for participation in Information Society। Alexandria Proclamation (2005): "Beacons of the Information Society: The Alexandria Proclamation on IL and Lifelong Learning"। High-Level Colloquium at Bibliotheca Alexandrina (Egypt)। Organized by IFLA + UNESCO। Key: IL + Lifelong Learning = Twin Pillars। "Beacons" metaphor = Lighthouse of Alexandria। Libraries = Beacons। Difference: Prague = IL as Right। Alexandria = IL + Lifelong Learning together।
Q6. Wilson's ISB Model क्या है? यह सबसे अधिक Cited क्यों है?+
T.D. Wilson's Information Behaviour Model (1981, Revised 1996) = Information Seeking Behaviour का सबसे Comprehensive और Most Cited Model। 1981 Model: Information Need → Information Seeking → Information Use + Barriers (Personal, Role, Environment)। 1996 Revision: Cognitive model integrate किया। Activating Mechanism = Stress/Cope। Intervening Variables। Information Behaviour का Nested Model। क्यों Most Cited: (1) General/Universal — किसी भी Context में applicable। (2) Barriers concept = Library Design में Help। (3) Need → Seeking → Use = Clear Flow। (4) Revised for Digital Age। LIS Research में Foundation Model।
Q7. Zipf का Least Effort Principle क्या है? इसका Library Science में क्या अर्थ है?+
George Zipf (1949) — Principle of Least Effort = People naturally choose the path of Least Resistance / Least Effort when seeking information। Library Science में Implications: (1) Users Google prefer करते हैं — Easiest Source। (2) Users nearest Library use करते हैं। (3) Users पहले Free Resources try करते हैं। (4) Complex Databases less popular। (5) User-friendly OPAC more used। यही Explain करता है: Google = Free + Easy = Preferred। Complex Academic Databases = Training required = Less spontaneous use। Library Design Implication: Make it EASY। Self-checkout, Online Renewal, 24×7 Access, Mobile App — सभी Least Effort Principle को address करते हैं।
Q8. Big6 Model क्या है? यह किसके लिए बना है?+
Big6 Model (1988) = Michael Eisenberg और Robert Berkowitz द्वारा। Primary audience: School Students & School Librarians। Purpose: Students को Research Process Systematically सिखाना। 6 Steps: (1) Task Definition = Problem Define करो — What do I need? (2) Information Seeking Strategies = Best Sources Decide करो। (3) Location & Access = Sources Find करो। (4) Use of Information = Read, View, Listen, Extract। (5) Synthesis = Organise और Present करो। (6) Evaluation = Process और Product दोनों Evaluate करो। Super3 = Big6 का simplified version for Younger students (Plan, Do, Review)।
Q9. SCONUL Seven Pillars Model क्या है?+
SCONUL Seven Pillars of Information Literacy = Society of College, National and University Libraries (UK)। First Published 1999, Revised 2011। Higher Education Libraries के लिए। 7 Pillars: (1) Identify = Know what you need। (2) Scope = Know what you know already। (3) Plan = Construct strategies। (4) Gather = Find & access resources। (5) Evaluate = Review & assess। (6) Manage = Organise, apply, cite। (7) Present = Share, communicate, use ethically। 2011 Revision में Core Model = "Understanding, Skills and Abilities" के Core पर। Each Pillar = Novice to Expert progression। Used widely in UK Academic Libraries।
Q10. Ellis के ISB Model के 6 Behaviours कौन से हैं?+
David Ellis's Behavioural Model (1989) = Social Scientists के Information Seeking Behaviour पर Based। Later Extended to other groups। 6 Behaviours: (1) Starting = Initial Sources से शुरुआत — known authors, references। (2) Chaining = References Follow करना — Forward/Backward Chaining। (3) Browsing = Semi-directed Scanning — Journals, Shelves। (4) Differentiating = Sources की Quality और Relevance Filter करना। (5) Monitoring = Field में New Developments Track करना — Alerts, TOC। (6) Extracting = Source से Relevant Items Systematically Identify करना। Later Added: Verifying, Ending। Ellis's Model = Behavioural (Observable Actions), Not Sequential (any order)।
